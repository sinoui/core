## 不同形状展示

### 形状1

```tsx
import Fab from '@sinoui/core/Fab';

const TabShapedFirst = styled(Fab)`
  border-radius: 50% 0;
  padding: 0;
  width: 56px;
  height: 56px;
  font-size: 24px;
  .sinoui-fab-extended__ripple-layout {
    width: 56px;
    height: 56px;
    border-radius: 50% 0;
  }
  .sinoui-fab-extended__ripple {
    width: 56px;
    height: 56px;
    border-radius: 50% 0;
  }
`;

function FabDemo() {
  return (
    <TabShapedFirst extended>
      <MdAdd />
    </TabShapedFirst>
  );
}
```

效果如下：

<TabShapedFirst />
