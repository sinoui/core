## 底部应用栏使用场景

底部应用栏建议用在如下场景：

- 仅限移动设备
- 访问底部导航抽屉
- 在屏幕上有两到五个的动作(包括浮动操作按钮)

底部应用栏不建议用在如下场景：

- 应用中已经存在底部导航栏
- 屏幕中最多只有一个动作

## 底部应用栏模式

根据应用栏的浮动按钮的不同形态将底部应用栏分为以下几种模式：

- 浮动按钮居中(默认)
- 浮动按钮居右显示
- 没有浮动按钮
- 浮动按钮居中嵌入显示

### 浮动按钮居中

代码：

```tsx
import React from 'react';
import { ThemeProvider } from 'styled-components';
import { defaultTheme } from '@sinoui/theme';
import BottomAppBar from '@sinoui/core/BottomAppBar';
import NavigationIcon from '@sinoui/core/NavigationIcon';
import AppBarActions from '@sinoui/core/AppBarActions';
import IconButton from '@sinoui/core/IconButton';
import Dehaze from '@sinoui/icons/Dehaze';
import Favorite from '@sinoui/icons/Favorite';
import Bookmark from '@sinoui/icons/Bookmark';
import Add from '@sinoui/icons/Add';
import Fab from '@sinoui/core/Fab';

export default function Demo() {
  return (
    <BottomAppBar
      navigationIcon={
        <IconButton color="#fff">
          <Dehaze />
        </IconButton>
      }
      fab={
        <Fab color="#000">
          <Add />
        </Fab>
      }
      actionItems={[
        <IconButton color="#fff">
          <Favorite />
        </IconButton>,
        <IconButton color="#fff">
          <Bookmark />
        </IconButton>,
      ]}
    />
  );
}
```

运行效果：

<CardStoryLayout height={200}>
  <BottomAppBarDemo style={{ position: 'relative', alignSelf: 'flex-end' }} />
</CardStoryLayout>
