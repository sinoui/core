## 底部应用栏使用场景

底部应用栏建议用在如下场景：

- 仅限移动设备
- 访问底部导航抽屉
- 在屏幕上有两到五个的动作(包括浮动操作按钮)

底部应用栏不建议用在如下场景：

- 应用中已经存在底部导航栏
- 屏幕中最多只有一个动作

## 底部应用栏结构

![](https://user-images.githubusercontent.com/20694684/75869481-95121a00-5e44-11ea-861b-404f078ab9d0.jpg)

1. BottomAppBar-顶部应用栏容器
2. NavigationIcon-应用栏左侧图标按钮区域
3. Fab-浮动操作按钮
4. AppBarActions-应用栏可操作性区域

## 底部应用栏模式

根据应用栏的浮动按钮的不同形态将底部应用栏分为以下几种模式：

- 浮动按钮居中(默认)
- 浮动按钮居右显示
- 没有浮动按钮
- 浮动按钮居中嵌入显示

### 浮动按钮居中

代码：

```tsx
import React from 'react';
import { ThemeProvider } from 'styled-components';
import { defaultTheme } from '@sinoui/theme';
import BottomAppBar from '@sinoui/core/BottomAppBar';
import NavigationIcon from '@sinoui/core/NavigationIcon';
import AppBarActions from '@sinoui/core/AppBarActions';
import { MdDehaze, MdFavorite, MdBookmark } from 'react-icons/md';
import Icon from './Icon';
import Fab from './Fab';

export default function Demo() {
  return (
    <BottomAppBar
      navigationIcon={
        <Icon>
          <MdDehaze />
        </Icon>
      }
      fab={<Fab />}
      actionItems={[
        <Icon>
          <MdFavorite />
        </Icon>,
        <Icon>
          <MdBookmark />
        </Icon>,
      ]}
    />
  );
}
```

运行效果：

</CardStoryLayout>
