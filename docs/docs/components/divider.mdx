### 水平居中分隔线

默认为 16px 的水平间距

<CardStoryLayout>
  <Divider horizontal middle />
</CardStoryLayout>

### 代码

```tsx
<Divider horizontal middle />
```

通过 marginHorizontal 属性设置水平间距为 48px 的水平分隔线

<CardStoryLayout>
  <Divider horizontal middle marginHorizontal={48} />
</CardStoryLayout>

### 代码

```tsx
<Divider horizontal middle marginHorizontal={48} />
```
