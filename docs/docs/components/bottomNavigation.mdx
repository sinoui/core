title: BottomNavigation 底部导航栏
---

使用底部导航栏，您可以在应用程序的主要导航项之间跳转。

底部导航栏在屏幕下方显示三到五个导航项。 每一个导航项都由一个图标和一个可选文本标签表示。 当点击底部导航图标时，用户被切换到该图标关联的目标页面顶部。

import BottomNavigation from '@sinoui/core/BottomNavigation';
import BottomNavigationAction from '@sinoui/core/BottomNavigationAction';
import { MdRestore, MdFavorite, MdLocationOn, MdFolder } from 'react-icons/md';
import { ThemeProvider } from 'styled-components';
import { defaultTheme } from '@sinoui/theme';

## BottomNavigation

## 基本使用

```tsx
import BottomNavigation from '@sinoui/core/BottomNavigation';
import BottomNavigationAction from '@sinoui/core/BottomNavigationAction';

function SvgIconDemo() {
  return (
    <BottomNavigation
      onChange={(e, value) => console.log(value)}
      value="favorites"
    >
      <BottomNavigationAction
        label="Recents"
        value="recents"
        icon={<MdRestore />}
      />
      <BottomNavigationAction
        label="Favorites"
        value="favorites"
        icon={<MdFavorite />}
      />
      <BottomNavigationAction
        label="Nearby"
        value="nearby"
        icon={<MdLocationOn />}
      />
      <BottomNavigationAction
        label="Folder"
        value="folder"
        icon={<MdFolder />}
      />
    </BottomNavigation>
  );
}
```

效果如下：

<ThemeProvider theme={defaultTheme}>
  <BottomNavigation
    onChange={(e, value) => console.log(value)}
    value="favorites"
  >
    <BottomNavigationAction
      label="Recents"

      icon={<MdRestore />}
    />
    <BottomNavigationAction
      label="Favorites"
      value="favorites"
      icon={<MdFavorite />}
    />
    <BottomNavigationAction
      label="Nearby"
      value="nearby"
      icon={<MdLocationOn />}
    />
    <BottomNavigationAction label="Folder" value="folder" icon={<MdFolder />} />

</ThemeProvider>
