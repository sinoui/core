布局组件是基于 24 列的栅格布局，使用栅格布局可以轻松实现响应式 UI。
布局组件中的行(Row)和列(Column)必须一起使用，才能达到栅格布局效果。

## 默认状态的栅格布局

行(Row)组件不设置间隔(gutter)属性时，列(Column)之间是没有间隔的。

Wrapper.tsx

```tsx
import styled from 'styled-components';

const Wrapper = styled.div`
  background: rgb(41, 45, 62);
  padding: 16px 0;
  & .sinoui-column {
    margin-bottom: 16px;
    > div {
      padding: 16px;
      background: #fff;
      border-radius: 4px;
      box-shadow: ${(props) => props.theme.shadows[1]};
    }
  }
`;
export default Wrapper;
```

Demo.tsx

```tsx
import React from 'react';
import { ThemeProvider } from 'styled-components';
import { defaultTheme } from '@sinoui/theme';
import { Row, Column } from '@sinoui/core/Grid';
import Wrapper from './Wrapper';

export default function Demo() {
  return (
    <ThemeProvider theme={defaultTheme}>
      <Wrapper>
        <Row>
          <Column xs={12}>
            <div>col-xs-12</div>
          </Column>
          <Column xs={12}>
            <div>col-xs-12</div>
          </Column>
          <Column xs={12}>
            <div>col-xs-12</div>
          </Column>
          <Column xs={12}>
            <div>col-xs-12</div>
          </Column>
        </Row>
      </Wrapper>
    </ThemeProvider>
  );
}
```

### 两边留白

Row 组件 gutterType 属性的默认值为 between，即两边不留白。通过 gutterType = "around" 可以将间隔方式改为两边留白。

Demo.tsx

```tsx
import React from 'react';

import { defaultTheme } from '@sinoui/theme';
import { Row, Column } from '@sinoui/core/Grid';
import Wrapper from './Wrapper';

export default function Demo() {
  return (
    <ThemeProvider theme={defaultTheme}>
      <Wrapper>
        <Row gutterType="around" gutter={8}>
          <Column xs={12}>
            <div>col-xs-12</div>
          </Column>
          <Column xs={12}>
            <div>col-xs-12</div>
          </Column>
          <Column xs={12}>
            <div>col-xs-12</div>
          </Column>
          <Column xs={12}>
            <div>col-xs-12</div>
          </Column>
        </Row>
      </Wrapper>
    </ThemeProvider>
  );
}
```

import Wrapper from './Wrapper';

  return (
    <ThemeProvider theme={defaultTheme}>
      <Wrapper>
        <Row gutter={8}>
          <Column xs={12} sm={6} md={4} lg={3}>

          </Column>
          <Column xs={12} sm={6} md={4} lg={3}>

## 自动布局

- 如果 Column 组件没有设置任何 xs、sm、md、lg、xs 属性值,则每列平均分配剩余空间。
- 如果 Column 组件通过 style 属性或者 ClassName 设置了 width 样式，则列宽为设置的 width 值。

Demo.tsx

```tsx
import React from 'react';
import { ThemeProvider } from 'styled-components';
import { defaultTheme } from '@sinoui/theme';
import { Row, Column } from '@sinoui/core/Grid';
import { RowProps } from '@sinoui/core/Grid/Row';
import Wrapper from './Wrapper';

export default function Demo() {
  return (
    <ThemeProvider theme={defaultTheme}>
      <Wrapper>
        <Row gutter={8}>
          <Column style={{ width: '200px' }}>
            <div>width: 200px</div>
          </Column>
          <Column>
            <div>auto</div>
          </Column>
          <Column>
            <div>auto</div>
          </Column>
          <Column>
            <div>auto</div>
          </Column>
        </Row>
      </Wrapper>
    </ThemeProvider>
  );
}
```

运行效果：

<CardStoryLayout>
  <GridColumnWidth gutter={8} />
</CardStoryLayout>
