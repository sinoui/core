title: Paper 纸张
---

应用程序的的背景类似纸张一样平坦、不透明，而其行为也像纸张一样，能够改变大小、与其他纸张绑定或者调换顺序。

import Paper from '@sinoui/core/Paper';
import { ThemeProvider } from 'styled-components';
import { defaultTheme } from '@sinoui/theme';

## 基本使用

```tsx
import Paper from '@sinoui/core/Paper';

const Div = styled.div`
  width: 128px;
  height: 128px;
  margin: 8px;
`;

function PaperDemo() {
  return (
    <Paper>
      <Div>Paper Content</Div>
    </Paper>
  );
}
```

效果如下：

<ThemeProvider theme={defaultTheme}>
  <Paper>
    <div style={{ width: '128px', height: '128px', margin: '8px' }}>
      Paper Content
    </div>
  </Paper>
</ThemeProvider>

## 设置阴影高度

import Paper from '@sinoui/core/Paper';

  width: 128px;
  height: 128px;
  margin: 8px;
