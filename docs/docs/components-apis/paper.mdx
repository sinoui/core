应用程序的的背景类似纸张一样平坦、不透明，而其行为也像纸张一样，能够改变大小、与其他纸张绑定或者调换顺序。

## 引用方式

```tsx
import Paper from '@sinoui/core/Paper';
```

## API

Paper 组件属性：

| 参数名称  | 参数类型 | 默认值 | 说明           |
| --------- | -------- | ------ | -------------- |
| elevation | number   | 2      | 阴影高度       |
| square    | boolean  | false  | 是否显示方角   |
| outlined  | boolean  | false  | 轮廓模式       |
| className | string   |        | 添加自定义类名 |

Paper 组件继承所有 div 元素属性。

更多使用示例，请参考[Paper Demos](/docs/components/paper)
