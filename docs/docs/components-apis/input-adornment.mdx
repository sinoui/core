输入框装饰器。

## 属性

| 属性                 | 属性类型         | 默认值 | 说明                                                           |
| -------------------- | ---------------- | ------ | -------------------------------------------------------------- |
| position             | 'start' \| 'end' |        | 装饰器相对于输入框的位置                                       |
| disablePointerEvents | boolean          |        | 禁止点击事件。这个属性的作用是：点击装饰器会让输入框获取到焦点 |
| children             | React.ReactNode  |        | 组件的内容。通常是一个 IconButton 或者字符串                   |
| as                   | React.ReactType  | 'div'  | 根元素使用的组件                                               |

## 入门示例

```tsx
import React from 'react';
import { ThemeProvider } from 'styled-components';
import { defaultTheme } from '@sinoui/theme';
import TextInput from '@sinoui/core/TextInput';
import InputAdornment from '@sinoui/core/InputAdornment';

function Demo() {
  return (
    <ThemeProvider theme={defaultTheme}>
      <TextInput
        label="重量"
        endAdornment={<InputAdornment position="end">Kg</InputAdornment>}
      />
    </ThemeProvider>
  );
}
```

更多关于 InputAdornment 的使用，请参考 [文本输入框 | 带装饰器的输入框](../components/textInput#带装饰器的输入框)。
