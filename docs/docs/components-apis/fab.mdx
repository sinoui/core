浮动按钮。

## API

Fab组件属性：

| 参数名称    | 参数类型      |默认值  | 说明                                                         |
| ----------- | --------------- |--------------- | ------------------------------------------------------------ |
| children    | React.ReactNode | |子元素  |
| mini    | boolean|false | 较小显示  |
| extended    | boolean |false | 扩展样式  |
| color    | string |primary | 指定颜色 |
| disabled    | boolean | false|不可用状态  |
| ref    | React.Ref  || 指向根元素 |
| className   | string        |  | 自定义类名 |  

## 引用方式

```tsx
import Fab from '@sinoui/core/Fab';
```

更多示例，请参考[Fab Demos](/docs/components/fab)
