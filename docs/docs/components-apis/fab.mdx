## API

Fab组件属性：

| 参数名称    | 参数类型        | 说明                                                         |
| ----------- | --------------- | ------------------------------------------------------------ |
| children    | React.ReactNode | 子元素  |
| mini    | boolean | 较小显示  |
| extended    | boolean | 扩展样式  |
| color    | string | 指定颜色  |
| disabled    | boolean | 不可用状态  |
| ref    | React.Ref | 指向根元素  |
| className   | string          | 自定义类名 |
| style    | React.CSSProperties | 指定样式  |
