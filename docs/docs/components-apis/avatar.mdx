Avatar 组件是用来展示头像的。

## 引用方式

```tsx
import Avatar from '@sinoui/core/Avatar';
```

## API

Avatar组件属性：

| 参数名称    | 参数类型   | 默认值        | 说明                                                         |
| ----------- | --------------- | ------------------------ | ----------------------------------- |
| dense       | boolean         |  false         |    是否为密集模式   |
| color       | string          | actionDisabled | 头像背景色          |
| children    | React.ReactNode |                |指定图标内容         |
| className   | string          |                | 添加自定义类名      |   

更多使用示例，请参考[Avatar Demos](/docs/components/avatar)
