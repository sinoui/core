按钮组件。

## 属性

| 属性             | 属性类型 | 默认值  | 说明                                    |
| ---------------- | -------- | ------- | --------------------------------------- |
| outlined         | boolean  | false   | 是否为轮廓按钮                          |
| raised           | boolean  | false   | 是否为容器按钮                          |
| disableElevation | boolean  | false   | 是否禁用海拔高度,在 raised 状态下起作用 |
| color            | string   | primary | 指定按钮颜色                            |

除上述属性外，Button 组件拥有 [BaseButton](./base-button) 的所有属性。

## 入门示例

```tsx
import React from 'react';
import styled, { ThemeProvider } from 'styled-components';
import { defaultTheme } from '@sinoui/theme';
import Button from '@sinoui/core/Button';

function Demo() {
  return (
    <ThemeProvider theme={defaultTheme}>
      <>
        <Button>default</Button>
        <Button outlined>default</Button>
        <Button raised>default</Button>
      </>
    </ThemeProvider>
  );
}
```

更多关于 Button 的使用示例，请参考 [Button Demos](../components/button)。
