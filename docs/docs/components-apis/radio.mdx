Radio 单选框组件。

## 引用方式

```tsx
import Radio from '@sinoui/core/Radio';
```

## API

Radio组件属性：

| 参数名称    | 参数类型   | 默认值        | 说明                                                         |
| ----------- | --------------- | ------------------------ | ----------------------------------- |
| color  | string | primary |指定颜色    |
| checked   | boolean  | false |是否选中   |
|  disabled  | boolean  | false |是否不可用   |
|  readOnly  | boolean  | false |是否只读   |
|  name  | string  |  |名称   |
|  value  | string  |  |单选框的值   |
|  onChange  | (event: React.ChangeEvent) => void  |  |值发生状态变化时的回调函数   |
|  onClick  | (event: React.MouseEvent) => void  |  |点击时调用的回调函数   |
| children    | React.ReactNode | |指定文字内容    |
| className   | string      |    | 添加自定义类名    |   

关于Radio的具体使用，请参考[Radio Demos](/docs/components/radio)
