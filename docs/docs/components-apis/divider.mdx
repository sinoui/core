## Divider 分割线

分隔线是对列表和布局中的内容进行分组的一条细线。

引入方式：`import Divider from '@sinoui/core/Divider';`

分割线有两种形式：

- 水平分割线
- 垂直分割线

其中水平分割线又分为：

- 全宽水平分割线
- 居中水平分割线
- 插入水平分割线

### 属性

| 属性名           | 默认值 | 类型                | 描述                                                                                 |
| ---------------- | ------ | ------------------- | ------------------------------------------------------------------------------------ |
| inset            | false  | Boolean             | 表示是否是插入分隔线。如果为 true，会有 72px 的左边距。                              |
| middle           | false  | Boolean             | 表示是否是中间分隔线。如果为 true，会有 16px 的水平边距。                            |
| horizontal       | false  | Boolean             | 表示是否为水平分隔线。默认值为 false,是垂直分割线。                                  |
| as               | 'hr'   | React.ReactType     | 指定根元素。                                                                         |
| marginLeft       |        | Number              | 指定左边距。                                                                         |
| marginRight      |        | Number              | 指定右边距。                                                                         |
| marginTop        |        | Number              | 指定上边距。                                                                         |
| marginBottom     |        | Number              | 指定下边距。                                                                         |
| marginVertical   |        | Number              | 指定垂直边距。                                                                       |
| marginHorizontal |        | Number              | 指定水平边距。                                                                       |
| flexItem         |        | Boolean             | 指定垂直分隔线在 flex 容器中使用。如果不指定，可能垂直分隔线的高度会被计算为 `0px`。 |
| style            |        | React.CSSProperties | 指定 css 样式。                                                                      |
| className        |        | string              | 指定 css 样式名。                                                                    |
