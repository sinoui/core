原生选择框组件。

## 引入方式

```ts
import NativeSelect from '@sinoui/core/NativeSelect';
```

## 属性

| 属性名称 | 属性类型           | 默认值 | 属性说明     |
| -------- | ------------------ | ------ | ------------ |
| multiple | boolean            | false  | 是否支持多选 |
| value    | string \| string[] |        | 值           |

除此之外，TextInput 继承了[TextInput](./text-input)的除了`multiline`、`minRows`、`maxRows`之外的所有属性。
