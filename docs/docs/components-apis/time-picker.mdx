日期区间选择组件。

## 引用方式

```ts
import TimePicker from '@sinoui/core/TimePicker';
```

## 属性说明

| 参数名称    | 参数类型                | 默认值 | 说明               |
| ----------- | ----------------------- | ------ | ------------------ |
| value       | string                  |        | 值                 |
| onChange    | (date?: string) => void |        | 值变化时的回调函数 |
| popperProps | PopperProps             |        | 给弹窗设置属性。   |
| hourStep    | number                  | 1      | 小时间隔           |
| minuteStep  | number                  | 1      | 分钟间隔           |
| minHour     | number                  | 0      | 最小小时数         |
| maxHour     | number                  | 23     | 最大小时数         |
| minMinute   | number                  | 0      | 最小分钟数         |
| maxMinute   | number                  | 59     | 最大分钟数         |

除上述属性外，TimePicker 还继承了[TextInput](./text-input)除了`value`， `multiline`， `minRows`， `maxRows`， `onChange`， 和 `multiple`之外的所有属性。

关于`TimePicker`的使用，请参考[日期选择示例](../components/datepicker)。
