## API

SvgIcon组件属性：

| 参数名称    | 参数类型        | 说明                                                         |
| ----------- | --------------- | ------------------------------------------------------------ |
| size   | number 、string |  设置图标大小。数字或者字符串。如果是数字，则单位是像素。   |
| viewBox   | string |   设置 svg 元素的 viewBox，默认为0 0 24 24  |
| color  | string |  设置颜色   |
| children    | React.ReactNode | 指定图标内容    |
| title  | string |  指定图标的标题   |
| className   | string          | 自定义类名
