## API

SvgIcon组件属性：

| 参数名称    | 参数类型        | 说明                                                         |
| ----------- | --------------- | ------------------------------------------------------------ |
| children    | React.ReactNode | 子元素    |
| size   | number 、string |  图标大小，为'inherit'继承父元素   |
| color  | string |  指定颜色   |
| as  | any |  指定根元素   |
| viewBox   | string |   svg可见区域设置，默认为0 0 24 24  |
| ref   | React.RefObject |   引用元素  |
| title  | string |  指定标题   |
| className   | string          | 自定义类名      
| style   | React.CSSProperties          | 指定样式
