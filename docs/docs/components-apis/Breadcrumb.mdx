Breadcrumb 面包屑导航组件。

## 引用方式

```tsx
import Breadcrumb from '@sinoui/core/Breadcrumb';
```

## API

Breadcrumb 组件属性：

| 参数名称  | 参数类型        | 默认值 | 说明         |
| --------- | --------------- | ------ | ------------ |
| children  | React.ReactNode |        | 子元素       |
| color     | string          |        | 指定背景颜色 |
| className | string          |        | 自定义类名   |
| separator | React.ReactNode |        | 分隔符       |
| component | React.ReactType |        | 指定元素属性 |

更多使用示例，请参考[Breadcrumb Demos](/docs/components/breadcrumb)。
