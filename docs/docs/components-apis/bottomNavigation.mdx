使用底部导航栏，您可以在应用程序的主要导航项之间跳转。

底部导航栏在屏幕下方显示三到五个导航项。 每一个导航项都由一个图标和一个可选文本标签表示。 当点击底部导航图标时，用户被切换到该图标关联的目标页面顶部。

## 引用方式

```tsx
import BottomNavigation from '@sinoui/core/BottomNavigation';
import BottomNavigationAction from '@sinoui/core/BottomNavigationAction';
```

## API

BottomNavigation 组件属性：

| 参数名称   | 属性类型        | 默认值 | 说明               |
| ---------- | --------------- | ------ | ------------------ |
| children   | React.ReactNode |        | 子元素             |
| showLabels | boolean         | true   | 是否显示所有标签   |
| onChange   | ()=>void        |        | 值变化时的回调函数 |
| value      | string          |        | value 值           |
| color      | string          |        | 指定背景颜色       |
| className  | string          |        | 添加自定义类型     |

BottomNavigationAction 组件属性：

| 参数名称  | 属性类型        | 默认值 | 说明         |
| --------- | --------------- | ------ | ------------ |
| label     | string          |        | 标签名称     |
| icon      | React.ReactNode |        | 图标         |
| showLabel | boolean         | false  | 是否显示标签 |
| value     | string          |        | value 值     |
| selected  | boolean         | false  | 是否选中     |

更多使用示例，请参考[BottomNavigation Demos](/docs/components/bottomNavigation)
