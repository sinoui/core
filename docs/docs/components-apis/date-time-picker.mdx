日期时间选择组件。

## 引用方式

```ts
import DateTimePicker from '@sinoui/core/DateTimePicker';
```

## DateRangePicker

| 参数名称       | 参数类型                          | 默认值 | 说明                                                                   |
| -------------- | --------------------------------- | ------ | ---------------------------------------------------------------------- |
| value          | string                            |        | 值                                                                     |
| onChange       | (date?: string) => void           |        | 值变化时的回调函数                                                     |
| isPc           | boolean                           |        | 是否是 pc 端设备                                                       |
| min            | string                            |        | 最小值                                                                 |
| max            | string                            |        | 最大值                                                                 |
| renderValue    | (value?: Date) => React.ReactNode |        | 指定自定义的值渲染函数。默认情况下，直接显示`value`属性值              |
| portal         | boolean                           |        | 是否将弹出内容以传送门的方式渲染。默认为`true`                         |
| popperProps    | PopperProps                       |        | 给弹窗设置属性                                                         |
| defaultYear    | number                            |        | 默认年份                                                               |
| defaultMonth   | number                            |        | 默认月份                                                               |
| showToday      | boolean                           | true   | 是否展示今天                                                           |
| startOfWeek    | 0 \| 1                            | 1      | 星期开始位置。`0`表示是星期日，`1`表示星期一。默认为`1`。              |
| minHour        | number                            | 0      | 最小小时数                                                             |
| maxHour        | number                            | 23     | 最大小时数                                                             |
| minMinute      | number                            | 0      | 最小分钟数                                                             |
| maxMinute      | number                            | 59     | 最大分钟数                                                             |
| hourStep       | number                            | 1      | 小时间隔                                                               |
| minuteStep     | number                            | 1      | 分钟间隔                                                               |
| skipMonthsView | boolean                           |        | 设置为`true`，则跳过月份选择。默认情况下，在桌面端不跳过，在移动端跳过 |

除上述属性外，DateTimePicker 还继承了[TextInput](./text-input)除了`value`， `multiline`， `minRows`， `maxRows`， `onChange`， 和 `multiple`之外的所有属性。

关于`DateTimePicker`的使用，请参考[日期选择示例](../components/datepicker)。
