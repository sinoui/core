CheckboxGroup 复选框组组件。

## 引用方式

```tsx
import CheckboxGroup from '@sinoui/core/CheckboxGroup';
```

## API

CheckboxGroup组件属性：

| 参数名称    | 参数类型   | 默认值        | 说明                                                         |
| ----------- | --------------- | ------------------------ | ----------------------------------- |
| color  | string | primary | 指定Checkbox的颜色    |
|  disabled  | boolean  | false |是否不可用   |
|  readOnly  | boolean  | false |是否只读   |
|  column  | boolean  | false | 设置是否纵向排列  |
|   gridLayout | boolean  | false |  网格对齐布局 |
|  columns  | number  | 3 | 按照几列做网格布局，默认为3列  |
|   enableSelectAll |  boolean | false| 是否支持全选 |
|   labelPosition |  'left' \| 'right' | 'right'| 标题显示位置 |
|  items  | React.ReactElement  | | 相当于children |
| children    | React.ReactNode | | 子元素    |
|  name  | string  |  |名称   |
|  value  | string  |  |复选框的值   |
|  onChange  | (event: React.ChangeEvent) => void  |  |复选框发生状态变化时调用的回调函数   |
|  onClick  | (event: React.MouseEvent) => void  |  |点击时调用的回调函数   |
| className   | string      |    | 添加自定义类名    |   

更多使用示例，请参考[CheckboxGroup Demos](/docs/components/checkboxGroup)
