## BottomAppBar 底部应用栏

底部应用程序栏组件提供对底部导航抽屉和最多四个操作的访问，包括浮动操作按钮。

引入方式：

```tsx
import BottomAppBar from '@sinoui/core/BottomAppBar';
```

组件结构：

```tsx
<BottomAppBar
  navigationIcon={
    <IconButton color="#fff">
      <Dehaze />
    </IconButton>
  }
  fab={
    <Fab color="#000">
      <Add />
    </Fab>
  }
  actionItems={[
    <IconButton color="#fff">
      <Favorite />
    </IconButton>,
    <IconButton color="#fff">
      <Bookmark />
    </IconButton>,
  ]}
/>
```

### 属性

| 属性名         | 默认值 | 类型            | 描述                                             |
| -------------- | ------ | --------------- | ------------------------------------------------ |
| insertFab      | false  | boolean         | 设置为 true，则 Fab 在底部应用栏居中且嵌入位置。 |
| endFab         | false  | boolean         | 设置为 true，则 Fab 在底部应用栏最右侧显示。     |
| navigationIcon |        | React.ReactNode | 底部应用栏左侧图标按钮                           |
| actionItems    |        | React.ReactNode | 底部应用栏可操作元素                             |
| fab            |        | React.ReactNode | 底部应用栏浮动操作按钮                           |
