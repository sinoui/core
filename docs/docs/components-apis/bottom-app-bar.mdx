## BottomAppBar 底部应用栏

底部应用程序栏组件提供对底部导航抽屉和最多四个操作的访问，包括浮动操作按钮。

引入方式：`import BottomAppBar from '@sinoui/core/BottomAppBar';`

### 属性

| 属性名    | 默认值 | 类型    | 描述                                             |
| --------- | ------ | ------- | ------------------------------------------------ |
| insertFab | false  | boolean | 设置为 true，则 Fab 在底部应用栏居中且嵌入位置。 |
| endFab    | false  | boolean | 设置为 true，则 Fab 在底部应用栏最右侧显示。     |
