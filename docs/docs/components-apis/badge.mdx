Badge 徽章组件。

徽章会在其子项的右上角生成一个小徽章。

## 引用方式

```tsx
import Badge from '@sinoui/core/Badge';
```

## API

Badge组件属性：

| 参数名称    | 参数类型   | 默认值        | 说明                                                         |
| ----------- | --------------- | ------------------------ | ----------------------------------- |
| children       | React.ReactNode         |           |    子元素   |
| color       | string          | parimary | 指定颜色          |
| dot         | boolean    |      false          |设置为圆点形式         |
| count         | number    |      0          |指定数字        |
| overflowCount         | number    |                |指定封顶数值         |
| showZero         | boolean    |      false          |设置数字为0时显示         |
| badgeContent         |   React.ReactNode  |                |设置自定义badge的内容        |
| title         | string    |                |设置鼠标悬浮时显示的文字         |
| anchorOrigin         | { horizontal: 'left' \| 'right'; vertical: 'bottom' \| 'top' }    |      { vertical: 'top', horizontal: 'right'}          |设置徽标的显示位置         |
| className   | string          |                | 添加自定义类名      |   

更多使用示例，请参考[Badge Demos](/docs/components/badge)
