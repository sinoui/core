CheckboxGroup 复选框组组件。

## 引用方式

```tsx
import CheckboxGroup from '@sinoui/core/CheckboxGroup';
```

## API

CheckboxGroup 组件属性：

| 参数名称        | 参数类型                          | 默认值  | 说明                               |
| --------------- | --------------------------------- | ------- | ---------------------------------- |
| color           | string                            | primary | 指定 Checkbox 的颜色               |
| disabled        | boolean                           | false   | 是否不可用                         |
| readOnly        | boolean                           | false   | 是否只读                           |
| column          | boolean                           | false   | 设置是否纵向排列                   |
| columns         | number                            |         | 按照几列做纵向布局                 |
| enableSelectAll | boolean                           | false   | 是否支持全选                       |
| labelPosition   | 'left' \| 'right'                 | 'right' | 标题显示位置                       |
| items           | CheckboxGrouopItem[]              |         | 指定可选项。也可以通过子元素指定   |
| children        | React.ReactNode                   |         | 子元素                             |
| name            | string                            |         | 名称                               |
| value           | string                            |         | 复选框的值                         |
| onChange        | (value: string[]) => void         |         | 复选框发生状态变化时调用的回调函数 |
| onClick         | (event: React.MouseEvent) => void |         | 点击时调用的回调函数               |
| className       | string                            |         | 添加自定义类名                     |

CheckboxGrouopItem 接口：

| 属性名 | 类型                | 说明                                         |
| ------ | ------------------- | -------------------------------------------- |
| id     | string \| undefined | 选项 id。可以为空。用来给元素添加 key 属性。 |
| value  | any                 | 选项值。                                     |
| label  | React.ReactNode     | 选项标签。                                   |

## 用法

```tsx
import CheckboxGroup from '@sinoui/core/CheckboxGroup';
import Checkbox from '@sinoui/core/Checkbox';

const Demo1 = () => {
  const [value, setValue] = useState(['1']);

  return (
    <CheckboxGroup value={value} onChange={setValue}>
      <Checkbox value="1">选项1</Checkbox>
      <Checkbox value="2">选项2</Checkbox>
      <Checkbox value="3">选项3</Checkbox>
    </CheckboxGroup>
  );
};
```

或者：

```tsx
import CheckboxGroup from '@sinoui/core/CheckboxGroup';
import Checkbox from '@sinoui/core/Checkbox';

const Demo2 = () => {
  const [value, setValue] = useState(['1']);

  return (
    <CheckboxGroup
      value={value}
      onChange={setValue}
      items={[
        { value: '1', label: '选项1' },
        { value: '2', label: '选项2' },
        { value: '3', label: '选项3' },
      ]}
    ></CheckboxGroup>
  );
};
```

关于 CheckboxGroup 的具体使用，请参考[CheckboxGroup Demos](../components/checkbox-group)
