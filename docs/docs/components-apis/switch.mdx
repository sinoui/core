开关是改变单个设置状态打开或关闭的控件。

## 属性

| 属性      | 属性类型                                               | 默认值             | 说明               |
| --------- | ------------------------------------------------------ | ------------------ | ------------------ |
| color     | string                                                 | primary            | 指定颜色           |
| className | string                                                 | sinoui-base-button | 自定义样式类名称   |
| checked   | boolean                                                | false              | 是否是选中状态     |
| disabled  | boolean                                                | false              | 是否是不可用状态   |
| value     | string                                                 |                    | 值                 |
| onChange  | `(event: React.ChangeEvent<HTMLInputElement>) => void` |                    | 值变更时的回调函数 |

## 简单使用

```tsx
import Switch from '@sinoui/core/Switch';

function Demo() {
  const [checked, setChecked] = useState(false);

  const handleChange = useCallback((event) => {
    setChecked(event.target.checked);
  }, []);

  return <Switch checked={checked} value="switch1" onChange={handleChange} />;
}
```

更多关于 Switch 的使用，请参考 [Switch](../components/switch)。
