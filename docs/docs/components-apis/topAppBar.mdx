## AppBar

AppBar 组件主要用来展示和当前页面有关的信息和操作，是一个承载应用栏图标、标题等内容的容器。

引入方式：`import AppBar from '@sinoui/core/AppBar';`

### 属性

| 属性名    | 默认值 | 类型    | 描述                                                          |
| --------- | ------ | ------- | ------------------------------------------------------------- |
| prominent | false  | boolean | 设置为`true`，表示是突出模式下的应用栏。                      |
| dense     | false  | boolean | 设置为`true`，表示是密集模式下的应用栏。                      |
| short     | false  | boolean | 设置为`true`，表示是收缩模式下的应用栏。                      |
| fixed     | false  | boolean | 设置为 true，页面发生滚动时，应用栏位置不变，否则随页面滚动动 |

## AppBarTitle

AppBarTitle 组件用来展示应用栏的标题。

标准模式的应用栏中标题文字不是很多，若标题内容过多需要换行的，推荐使用突出模式的应用栏。

引入方式：`import AppBarTitle from '@sinoui/core/AppBarTitle';`

### 属性

无

## NavigationIcon

NavigationIcon 组件是用来承载应用栏最左侧图标按钮的容器。

引入方式：`import NavigationIcon from '@sinoui/core/NavigationIcon';`

### 属性

无

## AppBarActions

AppBarActions 组件使用来承载应用栏可操作性按钮、图标等的容器。

### 属性

无

### 代码

```tsx
import AppBarActions from '@sinoui/core/AppBarActions';

import { MdDehaze, MdFavorite, MdBookmark } from 'react-icons/md';

export default function Demo(props: {
  prominent?: boolean;
  dense?: boolean;
  fixed?: boolean;
  short?: boolean;
}) {
  return (
    <AppBar>
      <NavigationIcon>
        <Icon>
          <MdDehaze />
        </Icon>
      </NavigationIcon>
      <AppBarTitle>标题</AppBarTitle>
      <AppBarActions>
        <Icon>
          <MdFavorite />
        </Icon>
        <Icon>
          <MdBookmark />
        </Icon>
      </AppBarActions>
    </AppBar>
  );
}
```
