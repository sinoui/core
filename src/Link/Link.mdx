# Link（链接）

链接（Links）组件允许您使用主题颜色和版面设计轻松自定义锚定元素。

## 初步认知

- [material-ui | Link](https://material-ui.com/zh/components/Links/)

## 引入方式

```tsx
import Link from '@sinoui/core/Link';
```

## 基本使用

```tsx
<Typography>
  <Link href="#" onClick={preventDefault}>
    Link
  </Link>
  <Link href="#" onClick={preventDefault} color="primary">
    color=primary
  </Link>
  <Link href="#" onClick={preventDefault} component="div">
    component=div
  </Link>
</Typography>
```

## 属性

| 参数名称  | 参数类型        | 默认值 | 说明       |
| --------- | --------------- | ------ | ---------- |
| children  | React.ReactNode |        | 子元素     |
| color     | string          |        | 指定颜色   |
| className | string          |        | 自定义类名 |
| ref       |                 |        | 指向根元素 |
| component | React.ReactType | a      | 指定根元素 |

- 链接（Links）组件构建在 版面设计（Typography） 组件之上。 您可以利用其属性。

- 当链接需要突出显示，使用 color="primary"

## 可访问性

[WAI-ARIA:](https://www.w3.org/TR/wai-aria-practices/#link)

- 当提供链接对应的内容时，避免使用泛泛的描述，比如“点击这里”或“跳转”之类的词语 相反的，请使用 具体详细的描述说明。

- For the best user experience, links should stand out from the text on the page。

- 如果一个链接没有赋予一个有意义的 href 值, 它应该用一个 <按钮> 元素表示。
