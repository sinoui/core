# BottomNavigation 底部导航栏

使用底部导航栏，您可以在应用程序的主要导航项之间跳转。

底部导航栏在屏幕下方显示三到五个导航项。 每一个导航项都由一个图标和一个可选文本标签表示。 当点击底部导航图标时，用户被切换到该图标关联的目标页面顶部。

## 初步认知

- [material-ui/BottomNavigation-组件](https://material-ui.com/zh/components/bottom-navigation/)
- [material-web-components/BottomNavigation demo](https://material.io/components/bottom-navigation/#specs)

## 用法

```tst
import BottomNavigation from '@sinoui/core/BottomNavigation';
import BottomNavigationAction from '@sinoui/core/BottomNavigationAction';
```

基本用法

```tsx
<BottomNavigation value={value} onChange={handleChange}>
  <BottomNavigationAction label="Recents" value="recents" icon={<RestoreIcon />} />
  <BottomNavigationAction label="Favorites" value="favorites" icon={<FavoriteIcon />} />
  <BottomNavigationAction label="Nearby" value="nearby" icon={<LocationOnIcon />} />
  <BottomNavigationAction label="Folder" value="folder" icon={<FolderIcon />} />
</BottomNavigation>
```

## BottomNavigation组件属性

| 属性名称 | 默认值      | 属性描述                                                 |
| -------- | ----------- | -------------------------------------------------------- |
| children |             | 子元素                                             |
| showLabels |             | 是否显示所有标签名称                             |
| onChange |             | 点击事件                             |
| className |             | 添加自定义类型                                             |

## BottomNavigationAction组件属性

| 属性名称 | 默认值      | 属性描述                                                 |
| -------- | ----------- | -------------------------------------------------------- |
| showLabel |             | 是否显示标签名称                             |
| value     |               | value值                             |

### API 文档

描述 SvgIcon 组件。
